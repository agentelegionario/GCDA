Configurações individuais dos rastreadores

Você poderá controlar cada rastreador individualmente. Se o controle deslizante ao lado de um rastreador estiver cinza, o rastreador está ativo e rastreando você. Para bloquear este rastreador, mova o controle deslizante para a direita. O controle deslizante ficará verde e o rastreador estará sendo bloqueado.
